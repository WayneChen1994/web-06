js中的函数 基本函数[小驼峰式命名]： function 函数名(参数列表){ 函数体; return 表达式; // 可有可无 } 匿名函数 var 变量名 = function(参数列表){ 函数体; return 表达式; // 可有可无 } ------------------------------------------------------------------ 全局变量与局部变量： 局部变量：定义在函数中的变量只能在当前函数中使用。 全局变量：在script中或者js文件中并且在函数体之外定义的变量，我们在整个html中所有的js文件或者script标签中都可以使用。 【注意定义的位置或者链接的位置】
