1.array对象的创建 方式1： var arr1 = [1, 2, 3, 4]; 方式2： // 开辟一个长度为3的空间 var arr2 = new Array(3); 方式3： // 创建一个数组，并且将1,2,3存入数组 var arr3 = new Array(1, 2, 3); ------------------------------------------------------------------ 2.获取数组的长度 arr.length ------------------------------------------------------------------ 3.数组的拼接 arr.concat(arr2) ----------------------------------------------------------------- 4.字符串与数组之间的转换 arr2.join(" ") 功能：将数组中的内容以指定的字符串拼接，返回一个新的字符串 str.split(" ") 功能：将原字符串以指定的字符串切割，返回一个新的数组。 ------------------------------------------------------------------- 5.向数组的末尾添加元素 arr.push(obj) 将指定的对象添加到数组的末尾，返回数组的长度。 ------------------------------------------------------------------ 6.向数组的开头添加元素 arr.unshift(obj) 将指定的对象添加到数组的开头，返回数组的长度。 ------------------------------------------------------------------ 7.arr.pop() 功能：删除数组中最后一个元素，并且返回被删除的元素。 ------------------------------------------------------------------- 8.arr.shift() 功能：删除数组中第一个元素，并且返回被删除的元素。 ------------------------------------------------------------------- 9.arr.reverse() 功能：数组反转 ------------------------------------------------------------------- 10.arr.slice(start, end) 功能：数组的截取，取头去尾，取值范围[start, end)。 -------------------------------------------------------------------
